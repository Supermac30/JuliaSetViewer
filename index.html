Julia Set Viewer

Welcome to My Julia Set Viewer. Type in the equation to see the corrseponding fractal. The Julia Set of a function is defined to be the set of points such that repeated iteration under some starting complex value doesn't diverge to infinite. We can plot this on the complex plane below with the a white pixel where ever it diverges, and a black pixel where it converges. The darker the pixel the faster it converges. Look at my blog post on fixed points to see how this iteration works and a future post on cool fractals to try.
